# 🎨 Generador de Paletas de Colores

Aplicación web moderna e interactiva que genera colores aleatorios, Puedes elegir entre 6, 8 y 9 colores. Permite cambiar formatos de color y copiar valores fácilmente.

---

## 🚀 Características

- 🎲 Generación de colores aleatorios
- 🎚️ Selección de cantidad de colores (6, 8, 9)
- 🎨 Cambio de formato de color:
  - RGB
  - HSL

- 📋 Copiar color HEX al hacer clic
- 👀 Mostrar código HEX al pasar el cursor
- 🎯 Contraste automático (texto siempre legible)
- 📱 Diseño responsive (mobile-first)

---

## 🎮 Instrucciones de Uso

1. Abra el aplicativo en el siguiente enlace:
   _(https://wijimenezz.github.io/ProyectoM1_WilsonJimenez-/)_

2. Encontrará una pantalla inicial (**Hero**) con el botón **"Generar Colores"**.
   - Haga clic en este botón para acceder al generador de colores.

3. Una vez dentro del generador:
   - Seleccione la cantidad de colores que desea visualizar:
     - 6 colores
     - 8 colores
     - 9 colores

4. Seleccione el formato de color:
   - RGB
   - HSL

5. Haga clic en **"Generar Colores"** para crear una nueva paleta.
   - Se generarán colores aleatorios
   - Se mostrarán los códigos según el formato seleccionado

6. Interacciones adicionales:
   - 👀 Pase el cursor sobre un color para ver su código HEX
   - 📋 Haga clic sobre un color para copiar el código HEX automáticamente

---

💡 **Nota:**
El código HEX es la referencia principal y puede ser utilizado en cualquier proyecto web o de diseño.
